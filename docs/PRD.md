# Picture to Drawing — MVP 产品需求文档

**版本：** v0.1
**日期：** 2026-04-08
**状态：** 草稿

---

## 一、产品概述

### 1.1 产品定位
Picture to Drawing 是一款基于 AI 的图片风格转换工具网站，用户上传照片后，可以将其转换为手绘素描、铅笔画等艺术风格图像，并下载使用。

### 1.2 目标用户
- 普通用户：想把自己或家人的照片做成手绘风格留作纪念
- 创作者：需要素材参考图、手绘风格内容的设计师、插画师
- 社交媒体用户：制作特色头像、个人主页封面图

### 1.3 MVP 核心目标
- 验证用户对"照片转手绘"功能的付费意愿
- 快速上线，2-4 周内完成
- 最低成本运行，按需扩展

---

## 二、功能范围

### 2.1 MVP 包含功能（Must Have）

| 功能 | 描述 |
|------|------|
| 图片上传 | 支持 JPG/PNG，大小限制 10MB |
| 风格选择 | 提供 3 种风格：铅笔素描、水墨画、线稿 |
| AI 转换 | 调用 Replicate API 进行图像处理 |
| 结果预览 | 转换完成后展示对比图（原图 + 效果图） |
| 图片下载 | 支持下载效果图（PNG 格式） |
| 免费额度 | 每个访客每天免费转换 3 次 |

### 2.2 MVP 不包含功能（Out of Scope）

- 用户注册 / 登录系统
- 付费订阅功能
- 历史记录
- 批量处理
- 移动端 App
- 社交分享

## 三、用户流程

```
进入首页
    ↓
上传图片（拖拽或点击选择）
    ↓
选择绘画风格
    ↓
点击「开始转换」
    ↓
等待处理（显示进度提示，预计 10-30 秒）
    ↓
查看效果（左侧原图 / 右侧效果图对比）
    ↓
点击「下载」保存图片
    ↓
提示：「还想转换更多？今日还剩 X 次免费机会」
```

## 四、页面设计

### 4.1 首页（唯一页面，单页应用）

**顶部区域**
- 网站 Logo + 名称「Picture to Drawing」
- 简短说明文案：「上传你的照片，AI 秒变手绘风格」

**核心交互区域**
- 上传框（支持拖拽，显示上传图标和提示文字）
- 风格选择器（图标卡片形式，3 种风格）
- 「开始转换」按钮（上传图片后激活）

**结果展示区域**（转换完成后显示）
- 左右对比图（原图 vs 效果图）
- 下载按钮
- 「再转一张」按钮

**底部**
- 版权信息
- 剩余免费次数提示

### 4.2 状态说明

| 状态 | 显示内容 |
|------|---------|
| 初始态 | 上传引导 |
| 上传中 | 进度条 |
| 处理中 | 动画 + 「AI 正在绘制中...」|
| 成功 | 对比图 + 下载按钮 |
| 失败 | 错误提示 + 重试按钮 |
| 次数用尽 | 「今日免费次数已用完，明天再来」|

## 五、技术方案

### 5.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端 | Next.js (React) | 静态导出，部署到 Cloudflare Pages |
| 后端 | Cloudflare Workers | 无服务器，处理 API 调用 |
| AI API | Replicate — `flux-kontext-max` | 图像风格转换核心能力 |
| 存储 | 无（图片走内存） | 原图和结果图不持久化存储 |
| 限流 | Cloudflare KV | 记录访客 IP 每日使用次数 |
| 部署 | Cloudflare Pages + Workers | 全站部署 |

### 5.2 数据流

```
前端上传图片（base64）
    → Cloudflare Worker 接收
    → 检查 IP 使用次数（Cloudflare KV）
    → 调用 Replicate API
    → 轮询等待结果
    → 将结果图片 URL 返回前端
    → 前端展示并供用户下载
```

### 5.3 关键约束

- 单张图片处理时间：预计 15-40 秒
- Cloudflare Worker 最大执行时间：默认 30s（需开启 `waitUntil` 或升级到 Unbound 模式）
- 图片大小限制：上传 ≤ 10MB，输出根据 API 返回

## 六、成本预估

| 项目 | 费用 | 说明 |
|------|------|------|
| Cloudflare Pages | 免费 | 100,000 请求/天免费 |
| Cloudflare Workers | 免费 | 100,000 次/天免费 |
| Cloudflare KV | 免费 | 读写额度内免费 |
| Replicate API | ~$0.05/张 | 按实际调用计费 |
| **100 次转换/天** | **~$5/天** | 冷启动期参考 |

## 七、验收标准

| 功能 | 验收条件 |
|------|---------|
| 上传图片 | 支持 JPG/PNG，超出大小给出提示 |
| 风格转换 | 3 种风格均可正常调用，返回结果图 |
| 结果展示 | 原图和效果图正确对比显示 |
| 下载功能 | 点击下载可保存效果图到本地 |
| 次数限制 | 同一 IP 每天超过 3 次后提示限制 |
| 错误处理 | API 失败时显示友好提示，可重试 |
| 加载性能 | 首页加载 < 3 秒 |
